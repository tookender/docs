## Acceptable Formats
We accept the following formats when we ask for a color:
- #hex
- rgb(number, number, number)

Number can be from 0-255 or 0%-100%, hex can be a 6 digit hex code number (#FF00AA) or a 3 digit hex shortcut (#fff)
